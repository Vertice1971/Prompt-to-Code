# Quiz: Realism in Literature – Grade 10 (U.S. High School)

This HTML file provides an interactive quiz on **Realism and Naturalism in Literature**, adapted for Grade 10 English curriculum in the United States. It is fully self-contained and requires no external files or internet connection.

## ✅ Content

The quiz includes **a total of 12 carefully selected questions**, covering key aspects of Realism and Naturalism with one correct answer per item. Questions are stored inside the code as a JSON object.

## ⚙️ How It Works

- When the page loads, the user sees a "Start Quiz" button.
- A prompt allows the user to select how many questions to answer.
- Questions are randomly selected and displayed one by one.
- Each answer option can be clicked only once.
- The “Next” button advances to the next question.
- At the end, the number of correct answers is displayed.
- The quiz can be restarted from scratch.
